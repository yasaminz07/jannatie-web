# Icons

Place your Jannatie PWA icons here. Required sizes:

- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-192x192.png (required for Android install prompt)
- icon-256x256.png
- icon-512x512.png (required for splash screen)
- apple-touch-icon.png (180x180, for iOS)

Design: Blue (#0066CC) background, white "J" or Arabic "ج" mark, with orange accent.

Use a tool like https://www.pwabuilder.com/imageGenerator to generate all sizes from a single 512x512 source image.
